# 📖❤️ Histórias de Amor

**Histórias de Amor** é uma aplicação web interativa desenvolvida para casais eternizarem seus momentos especiais de forma única e personalizada. O sistema permite a customização de um hotsite exclusivo que é publicado automaticamente após a aprovação do pagamento.

## 💻 Sobre o Projeto

A ideia principal desta plataforma é oferecer uma experiência fluida onde o usuário pode acessar o site, fazer o upload e a edição de fotos do casal e montar uma página declarando o seu amor. Ao finalizar as edições, o usuário clica em "Criar Site Único". 

Neste momento, a aplicação exibe um checkout de pagamento integrado diretamente com a **API do Mercado Pago**. Após a confirmação da transação, o site exclusivo do casal é gerado e disponibilizado para ser compartilhado.

## ✨ Funcionalidades

- **Upload e Edição de Fotos:** Interface amigável para o usuário carregar e ajustar fotos do casal.
- **Geração Dinâmica de Página (Hotsite):** Criação de uma página web única e formatada com as fotos e textos inseridos pelo usuário.
- **Checkout de Pagamento:** Modal de cobrança transparente e seguro gerado via API do Mercado Pago.
- **Responsividade:** Layout adaptável, garantindo que a edição e a visualização fiquem perfeitas tanto no celular quanto no computador.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

- **HTML5:** Estruturação semântica da aplicação.
- **TypeScript:** Estilização responsiva e design de interface (UI).
- **JavaScript (ES6+):** Lógica de manipulação de imagens, eventos da página e validações.
- **API Mercado Pago:** Integração de gateway de pagamento para monetização do serviço.
